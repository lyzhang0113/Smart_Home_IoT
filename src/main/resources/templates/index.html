Smart Home

This project will develop a smart home device prototype along with a simple web user interface using Node-RED, ESP32, and various sensors, the team will demonstrate the use of a smart home system.

How does it work?

The team will make use of various sensors such as a temperature/humidity sensor, a light sensor, and a motion sensor, to collect necessary data to be displayed to the end-user through a web-UI and an on-board display.

System Structure Design

The team will make use of various sensors such as a temperature/humidity sensor, a light sensor, and a motion sensor, to collect necessary data to be displayed to the end-user through a web-UI and an on-board display. A detailed description of each hardware and software the device uses is shown on the right.

Hardware Design

  • ESP32
  • DHT11
  • PIR Motion Sensor
  • LCD Display
  • Photoresistor

Software Design

  • Web-framework (Java Spring Boot) for Web User-Interface
  • User Password are Encrypted
  • MySQL Database for Durable Data Storage
  • Mosquitto MQTT
  • Node-RED
  • Redis for caching user information
